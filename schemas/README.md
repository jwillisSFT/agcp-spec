# JSON Schema Definitions

Status: Normative  
Version: 0.9.0  

This directory contains the JSON Schema artifacts used by AGCP implementations.

These schemas define the canonical structure of governance envelopes, validation results, error responses, and other structured artifacts exchanged by AGCP systems.

---

# Purpose

The schemas serve three purposes:

1. Define canonical request and response structures.
2. Enable automated validation of AGCP envelopes.
3. Ensure cross-implementation interoperability.

---

# Schema Categories

The schemas include definitions for:

- action envelopes
- validation results
- error responses
- execution commit requests
- governance artifacts

---

# Versioning

Schemas are versioned alongside the AGCP specification.

Breaking schema changes require a **major version increment** of the AGCP specification.
